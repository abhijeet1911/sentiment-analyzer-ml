# AI Sentiment Analyzer 🤖

A lightweight, offline-capable sentiment analysis tool built using Python, Flask, and Scikit-learn.

## Features
- **Custom ML Model:** Built using Naive Bayes classifier (no paid APIs!).
- **Real-time Prediction:** Analyzes text sentiment (Positive/Negative/Neutral) instantly.
- **Confidence Scoring:** Shows how confident the model is with its prediction.
- **Responsive UI:** Clean chat-based interface.

## Tech Stack
- **Backend:** Flask (Python)
- **ML:** Scikit-learn (Naive Bayes)
- **Frontend:** HTML/CSS/JS

## How to Run
1. Clone this repo.
2. Install dependencies: `pip install flask scikit-learn`
3. Run the app: `python app.py`
4. Open `http://127.0.0.1:5000` in your browser.
